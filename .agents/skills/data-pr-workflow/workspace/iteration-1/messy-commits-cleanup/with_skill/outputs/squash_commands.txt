## Git Commit Cleanup Commands

### Interactive Rebase
```bash
git rebase -i main
```

In the editor, squash the messy commits (wip, fix tests):
```
pick abc1234 feat(dbt): add user_engagement model
squash def5678 wip - initial structure
squash ghi9012 add firebase event aggregation
squash jkl3456 fix tests
squash mno7890 wip - add metrics
squash pqr1234 fix tests
squash stu5678 add documentation
squash vwx9012 final cleanup
```

### Force Push
```bash
git push --force-with-lease origin DE-4521/add-user-engagement-model
```
