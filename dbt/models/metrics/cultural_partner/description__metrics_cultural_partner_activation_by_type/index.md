The `metrics_cultural_partner__activation_by_type` model provides aggregated key indicators related to pass Culture cultural partner activations. It is designed to be the central model providing cultural partners activation indicators for export, monitoring and gouvernance.

## Table Description

Each row represents the key indicators related to cultural partners activation on individual and collective part of the pass Culture, calculated for a specific month, a type of partner and geographic aggregation level. This models is created separately from the activation model because the different types of partner are not additive.
