# Table: User

This model exports user data for ministry use. It contains information about users including:
- Basic information (ID, activity, civility, school type)
- Status information (active status, current beneficiary status)
- Personal information (age, birth date)
- Account information (creation date, activation date, suspension reason)
- Deposit information (first deposit date and type, current deposit type)
- User seniority
