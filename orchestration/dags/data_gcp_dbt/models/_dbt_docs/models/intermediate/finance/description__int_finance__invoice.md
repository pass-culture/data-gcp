The `int_finance__invoice` table contains reimbursement receipts generated when wire transfers are confirmed.

## Business Context

Invoices are generated by the `generate_invoices` command after cashflow generation. Each invoice is linked to a bank account and summarises the wire transfers made. A CSV summary file is also deposited on Google Drive for accounting purposes.
