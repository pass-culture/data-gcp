{% docs description__metrics_eac__offer_booking %}

The `metrics_eac__offer_booking` model provides aggregated key indicators related to
collective (EAC) offers and bookings. It is designed to be the central model exposing
EAC offer and booking indicators for export, monitoring and governance.

{% enddocs %}

## Table Description

Each row represents key indicators calculated for a specific month, venue geographic aggregation
and institution geographic aggregation level.
