The `metrics_institution__activity` model provides aggregated key indicators related to pass Culture institutions on the collective part.
It is designed to be the central model providing institutions activity indicators for export, monitoring and gouvernance.

{% enddocs %}

## Table Description

Each row represents the key indicators related to institution activity and collective part of the pass Culture, calculated for a specific month, a sholar year and geographic aggregation level.
