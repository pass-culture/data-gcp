The `metrics_offer__collective` model provides aggregated key indicators related to collective offer creation by venue geography.
It is designed to be the central model providing collective offer creation indicators for export, monitoring and governance.

## Table Description

Each row represents the key indicators related to collective offer creation, calculated for a specific month and venue geographic level (department / region / city / EPCI).
