The `metrics_student__coverage` model provides aggregated key indicators related to student participation in EAC (Education Artistique et Culturelle).
It is designed to be the central model providing student coverage indicators for export, monitoring and governance.

## Table Description

Each row represents the key indicators related to student EAC participation, calculated for a specific month, school year and geographic aggregation level.
